##  Proje Hakkında
PHISHGUARD makine öğrenmesi tekniklerini kullanarak URL’leri ve e-posta içeriklerini analiz eden Python tabanlı bir phishing (oltalama) ve spam tespit sistemidir. 
 
Amaç zararlı web bağlantılarını ve spam/phishing e-postalarını tespit ederek kullanıcı güvenliğini artırmaktır.
